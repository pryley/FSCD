«SnIco Edit 2.2

SnIco Edit is an advanced icon editor with support for 2, 16, 256 and true colors as well as the new XP 32-bit with alpha transparency. It can import/export icons from/to GIF, BMP, DLL and EXE files and offers several drawing tools including a Text tool, a Shape tool, selection, fill and more. Icons can be rotated, flipped, moved and copied to the clipboard. A small preview window allows you to view the icon in real size as you are editing it.

Features:

  • Support for Windows icons and cursors: default sizes 16x16, 32x32 and 48x48. Also custom sizes are supported. All colordepths are supported including Windows XP alphablending.
  • Support for animated icon files (ANI files).
  • Importing from EXE, DLL, BMP, GIF, PNG, JPG and screen.
  • Exporting to BMP, GIF, PNG and JPG.
  • Many effects like: RGB/HLS adjusting, blurring, rotating, antialiasing.
  • Multi-language support
  • Multiple files open at the same time.
