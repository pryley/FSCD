Internet > Browsing > WinHTTrack

«WinHTTrack 3.33

WinHTTrack is an easy-use offline browser utility. It allows you to download a World Wide Web site from the Internet to a local directory, building recursively all directories, getting HTML, images, and other files from the server to your computer. WinHTTrack arranges the original site's relative link-structure. Simply open a page of the "mirrored" website in your browser, and you can browse the site from link to link, as if you were viewing it online. You can mirror several sites together so that you can jump from one to another. You can, also, update an existing mirror site, or continue an interrupted transfer. WinHTTrack is fully configurable, and has an integrated help system. The program is Open Source.
