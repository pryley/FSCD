«DVD43 3.5.2

DVD43 is the FREE version of DVD43 that integrates into Windows and overrides CSS copy-protection mechanisms and region limitations found on DVDs. Once it's installed, it runs in the background and you don't need to worry about it. You can use DVD43 to remove CSS protection on many popular DVD copying programs.

DVD43 only works on Win2K and WinXP. If you have Win98 or WinME, we suggest you either try DVD43 1.4, or use DVD Decrypter which allows you to either rip all the files of the DVD to your hard drive, or rip the DVD to an ISO file which you can then mount to a virtual drive with DAEMON Tools.
