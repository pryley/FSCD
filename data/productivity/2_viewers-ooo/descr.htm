«OpenOffice.org Viewers

Unfortunately, there is no official Viewer for the OpenOffice.org office suite, previously the only way to view documents saved with the open standard XML format (.sxw, .sxc, .sxi) was either to have the whole OpenOffice.org or StarOffice suite installed on the target computer, or to save them in the Microsoft Office (.doc, .xls, .ppt) format. These are viewers created with java, they are not offical viewers and are still under development.

The Java Runtime Enviroment must be installed to use the OpenOffice.org viewers.
