«Total Commander 6.52

Total Commander (formerly known as Windows Commander) is one of the best file managers available. It's a two panel file manager, using two window panes side by side to show different directories. This makes it easy to compare directories, to see which files are different in which directories, to copy files back and forth, etc. Folder tabs help you quickly switch between multiple directories, preserving display modes for each tab.

The built-in file viewer (Lister) lets you view files of ANY SIZE in hex, binary or text format, including RTF, HTML, and BMP files. You can even view files inside archives! The lister can be extended with plugins to view almost every file format including images, multimedia, documents; it can be used as a quick view panel (Ctrl+Q) to preview files before you open them.

Archives are handled like directories, making it easy to copy files to and from archives. It has an internal ZIP-compatible packer and internal unpackers for ZIP, ARJ, LZH, TAR, GZ, CAB, RAR and ACE formats. It can provide full support for almost every archive format through external packers and packer plugins.

Use the command line to start programs with parameters, simply type the program name or press CTRL+ENTER or CTRL+SHIFT+ENTER to feed the selected file to the command line. Syncronize whole folder structures, for instance between two computers or between your own computer and a FTP-server on the Net. Rename multiple files at once with powerful rename options using the Multi-Rename tool. Use the enhanced Search to find files with a specific search pattern, size, date or contents, with full text search in any file across multiple drives, even inside archives. Search for duplicate files, Split/Combine big files, Encode/Decode files in UUE, XXE and MIME format, Create/Verify CRC Checksums, and more.

Total Commander contains a built-in FTP client. The FTP client not only allows uploading and downloading files, but also supports the direct transfer of files from one remote server to another. You can even view and edit remote files without explicitly downloading them to your local machine.

The interface is highly configurable, you can adjust it to however suits your needs best. You can customize the button bar (use multiple button bars), main menu, and start menu (User-defined commands) to place your frequently used programs and commands.
